# Operator Dasar

Sekarang mari kita lanjutkan pembelajaran Javascript dengan belajar _operator dasar_.

## Apa itu Operator?

Operator pada dasarnya digunakan untuk melakukan perubahan(manipulasi) terhadap satu atau lebih nilai(value). Beberapa jenis operator antara lain: Operator _Aritmatika_, _Logika_, _Presedence_, _Assignment_, dan masih banyak lagi.

### Operator Aritmatika

Operator Aritmatika merupakan operasi matematika yang biasa kita lakukan seperti tambah, bagi, kurang, kali.

#### Simbol

| Operator    | Simbol | Penjelasan                                                        |
| ----------- | ------ | ----------------------------------------------------------------- |
| Pertambahan | +      | Digunakan untuk melakukan penjumlahan satu atau lebih nilai.      |
| Pengurangan | -      | Digunakan untuk melakukan pengurangan satu atau lebih nilai.      |
| Perkalian   | \*     | Digunakan untuk melakukan perkalian satu atau lebih nilai.        |
| Pembagian   | /      | Digunakan untuk melakukan pembagian antara satu atau lebih nilai. |

#### Contoh Penggunaan

```javascript
const tambah = 1 + 2; //Output: 3

const kurang = 1 - 2; //Output: -1

const kali = 1 * 2; //Output: 2

const bagi = 2 / 1; //Output: 1
```

### Operator Logika

Operator Logika atau _Logical Operator_ merupakan operasi yang digunakan dengan tujuan untuk menghasilkan sebauh nilai `true` dan `false`.

#### Simbol

| Operator | Simbol | Istilah resmi | Penjelasan                                                   |
| -------- | ------ | ------------- | ------------------------------------------------------------ |
| Dan      | &&     | And           | Digunakan untuk melakukan penjumlahan satu atau lebih nilai. |
| Atau     |        | Or            | Digunakan untuk melakukan pengurangan satu atau lebih nilai. |
