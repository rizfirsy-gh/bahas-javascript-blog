# Membuat Environment Belajar

## Install Node.js

Dimulai dari download dan install **node js** dulu.

- Download [Node.js](https://nodejs.org/en/download/)
  - Ketika install nodejs, saya menyarankan untuk menggunakan yang versi LTS(Long Term Support).

## Install Code Editor

Kemudian download dan install **code editor** dulu.
Code editor yang saya rekomendasikan adalah [Visual Studio Code](https://code.visualstudio.com/). Saya akan menyebutnya **vscode** diseluruh tutorial ini.

## Install VSCode Extension

Sekarang saya asumsikan kamu sudah berhasil menginstall vscode, step selanjutnya adalah install extension biar enak pas ngoding javascript.

`View > Extensions`
Shortcut: `Ctrl + Shift + X`

## Membuat folder belajar

Buatlah satu **folder baru** khusus untuk belajar javascript.
Disini saya akan memberi nama `belajar-javascript`.

Cara membuat folder lewat terminal windows

```bash
 mkdir belajar-javascript
```

Masuk ke folder lewat terminal windows

```bash
 cd belajar-javascript
```
