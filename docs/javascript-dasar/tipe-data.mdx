# Tipe Data JavaScript

Dalam bahasa pemrograman apapun, setiap value memiliki tipe yang beragam tergantung jenis data seperti apa yang kita simpan ke sebuah variable.

:::info
Pada pemrograman Javascript, sebuah value dapat berupa antara _object(non-primitive)_ atau _primitive value_.
:::

Contoh value berupa **object(non-primitive)** itu seperti ini:

```javascript
const namaLengkap = {
  namaDepan: "Cinthya",
  namaBelakang: "Rizky",
};
```

Contoh value berupa **primitive** itu seperti ini:

```javascript
const namaLengkap = "Cinthya Rizky";
```

## Cara mengetahui jenis tipe data

Jika kamu ingin mengetahui tipe data dari sebuah variable atau value, kamu bisa menggunakan keyword `typeof`.

```javascript
const namaLengkap = "Cinthya Rizky";

console.log(typeof namaLengkap);
//Output: string
```

## Primitive Value

Primitive value hanya bisa menyimpan satu buah value. Ada 7 jenis primitive value:

| No. | Tipe data       | Deskripsi                                                                                                                       |
| --- | --------------- | :------------------------------------------------------------------------------------------------------------------------------ |
| 1.  | **String**      | Berupa huruf, kata, atau kalimat yang ditandai dengan tanda petik satu(' ') atau dua(" ").                                      |
| 2.  | **Number**      | Berupa bilangan bulat atau desimal. Misalkan ada number `23` maka dia sebenarnya adalah `23.0`.                                 |
| 3.  | **Boolean**     | Digunakan untuk pengkondisian yang menghasilkan antara ya atau tidak, dengan value `true` untuk ya dan `false` untuk tidak.     |
| 4.  | **Undefined**   | Tidak ada value.                                                                                                                |
| 5.  | **Null**        | Value spesial yang berarti 'tidak ada', 'kosong', atau 'tidak diketahui'.                                                       |
| 6.  | **Symbol**(ES6) | Symbol merupakan fitur baru yang dirilis pada Javascript ES6. Symbol digunakan untuk menyimpan anonymous dan unique value.      |
| 7.  | **BigInt**      | Digunakan untuk menyimpan bilangan bulat(integer) yang memiliki jumlah digit melewati batas aman jumlah digit tipe data Number. |

Contoh penggunaan tipe data primitive:

```javascript
const nama = "Cinthya Rizky";
const usia = 12;
const tinggiBadan = 1.8;
const perempuan = true;
const hobi = undefined;
const alergi = null;
const mySymbol = Symbol("uniqueIDblablabla");
const bigInteger = 234567890123456789012345678901234567890;

console.log(
  typeof nama, //string
  typeof usia, //number
  typeof tinggiBadan, //number
  typeof perempuan, //boolean
  typeof hobi, //undefined
  typeof alergi, //object(bug di Javascript: null dianggap object value)
  typeof mySymbol, //Symbol
  typeof bigInt //bigint
);
```

## Object Value(non-primitive)

Object digunakan untuk menyimpan sekumpulan data dengan format pasangan _key-value_.
Value yang berupa object juga disebut dengan _non-primitive_ value.
Dengan object value, kita dapat menyimpan lebih dari satu values.

```javascript
//object
const Rizky = {
  namaLengkap: "Cinthya Rizky",
  umur: 22,
};

//array
const olahraga = [
  "Sepakbola",
  "Futsal",
  "Badminton",
  "Tenis",
  "Voli",
  "Basketball",
];
```

:::note
Perlu diingat, (di Javascript) tipe data yang bukan primitive maka dia adalah tipe data object.
:::
