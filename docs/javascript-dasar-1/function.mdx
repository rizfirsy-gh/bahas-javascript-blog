# Function

Function merupakan satu kesatuan instruksi program yang bertugas untuk menjalankan satu tugas spesifik. Untuk menjalankan program didalam function kita hanya perlu memanggil nama function tersebut.

```javascript
//membuat function
function tambah(a, b) {
  return a + b;
}

//memanggil function
tambah(1, 2); //output = 3
```

### Struktur `Function`:

| Struktur  | Deskripsi                                                                                                                                                                      |
| --------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Nama      | Nama method. Aturan penulisan sama dengan variable.                                                                                                                            |
| Parameter | Yang dibutuhkan method, ditulis didalam kurung/_parentheses_ `()`. Jika method tidak membutuhkan parameter, maka tidak perlu memberi apapun didalam kurung _parentheses_ `()`. |
| Body      | Ini adalah code block, dimana kita akan menulis semua kodingan kita yang berkaitan dengan tugas method tersebut.                                                               |
| return    | Value yang harus dikembalikan oleh method. return ada didalam body.                                                                                                            |
