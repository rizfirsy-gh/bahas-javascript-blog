# Input dan Output

Input dan output merupakan sebuah komunikasi antara user dengan aplikasi. Input adalah apa yang user berikan/masukan kepada aplikasi. Sedangkan output adalah apa yang aplikasi berikan/hasilkan kepada user.

## Input

Javascript dapat menerima input dalam beberapa cara, antara lain:

- Dari tag HTML `input`
- Menggunakan `prompt()` method
  :::info
  Blog cara meminta input user menggunakan `prompt` bisa dibaca [disini]
  :::
