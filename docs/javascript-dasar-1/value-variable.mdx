## Apa itu value dan variable?

Value dan variable ibarat sendal kiri dan kanan, mereka ada dan selalu <BeautySentence>bersama</BeautySentence> untuk saling melengkapi! Meski begitu, keduanya memiliki definisi masing-masing loh. Seperti apa? Yuk kita pahami penjelasan dibawah ini!

## Penjelasan Dasar

**Variable** bisa dibayangkan sebagai sebuah kardus yang bisa kamu gunakan untuk menyimpan benda-benda. Kardus tersebut bisa kamu beri label untuk memberi penjelasan apa isi dari kardus tersebut. Benda-benda yang disimpan didalam kardus tersebut disebut **value**.

Sebagai contoh: Ada deklarasi sebuah variable `kardusBuku = ["Buku Gambar", "Buku Tulis", "Buku Nota"]`, maka `kardusBuku` adalah sebuah `variable` yang diberi label "kardusBuku"(karena didalam kardus tersebut kita simpan beberapa jenis buku), sedangkan `["Buku Gambar", "Buku Tulis", "Buku Nota"]` adalah sebuah `value` atau isi dari kardusBuku.

## 3 Jenis Keyword Variable

Javascript punya 3 Keyword untuk mendeklarasikan sebuah variable, yaitu: `var` , `let` , dan `const`.

**`var` digunakan untuk variable yang valuenya kemungkinan akan berubah.**
`var` juga bisa dideklarasikan tanpa memberi value.

```javascript
//deklarasi dengan value
var nama = "Cinta";

//deklarasi tanpa value
var usia;

//memberi value kepada variable
usia = 12;
```

**`let` digunakan untuk variable yang valuenya kemungkinan akan berubah.**
`let` juga bisa dideklarasikan tanpa memberi value.

```javascript
//deklarasi dengan value
let nama = "Cinta";

//deklarasi tanpa value
let usia;

//memberi value kepada variable
usia = 12;
```

**`const` digunakan untuk variable yang memiliki value tetap(konstanta) atau tidak dapat diganti.**
Dengan menggunakan `const` kamu wajib memberi value.

```Javascript
    const nama = "Cinta";
```

## Penamaan variable yang baik dan benar

Pemberian nama sebuah variable ada aturannya dengan tujuan supaya dapat dipahami oleh kita sendiri dan teman developer lainnya. Mari kita bahas seperti apa _best practice_-nya.

1. Menggunakan camelCase
   - camelCase adalah teknik penamaan dengan memberi kapital hanya dari kata kedua dari nama variablenya.
2. Tidak diawali dengan huruf kapital
3. Tidak boleh diawali dengan angka
4. Dapat merepresentasikan valuenya
5. Tidak terlalu panjang
6. Tidak terlalu singkat.
