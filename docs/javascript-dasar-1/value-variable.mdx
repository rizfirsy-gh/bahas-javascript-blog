# Value dan Variable

## Penjelasan Dasar

**Variable** bisa dibayangkan sebagai sebuah kardus yang bisa kamu gunakan untuk menyimpan benda-benda. Kardus tersebut bisa kamu beri label untuk memberi penjelasan apa isi dari kardus tersebut. Benda-benda yang disimpan didalam kardus tersebut disebut **value**.

Sebagai contoh: Ada deklarasi sebuah variable `kardusBuku = ["Buku Gambar", "Buku Tulis", "Buku Nota"]`, maka `kardusBuku` adalah sebuah `variable` yang diberi label "kardusBuku"(karena didalam kardus tersebut kita simpan beberapa jenis buku), sedangkan `["Buku Gambar", "Buku Tulis", "Buku Nota"]` adalah sebuah `value` atau isi dari kardusBuku.

## 3 Jenis Keyword Variable

JavaScript punya 3 Keyword untuk mendeklarasikan sebuah variable, yaitu: `var` , `let` , dan `const`.
`var` dianjurkan untuk tidak digunakan lagi, jadi saya hanya akan menjelaskan `let` dan `const` saja.
