# Membuat Environment Belajar

## Install Browser

Tahap pertama adalah memastikan di laptop/komputer kamu ada browser. Saya merekomendasikan untuk menggunakan [Google Chrome](https://www.google.com/chrome/), tapi jika menggunakan yang lain juga tidak masalah.

## Install Node.js

Setelah itu download dan install [Node.js](https://nodejs.org/en/download/) dulu. **Node.js** ini berguna supaya kita bisa menjalankan file javascript kita di laptop/komputer kita sendiri(local).

- Download [Node.js](https://nodejs.org/en/download/)
  - Ketika install nodejs, saya menyarankan untuk menggunakan yang versi LTS(Long Term Support).

## Install Code Editor

Kemudian download dan install **code editor** dulu.
Code editor yang saya rekomendasikan adalah [Visual Studio Code](https://code.visualstudio.com/). Saya akan menyebutnya **vscode** diseluruh tutorial ini.

## Install VSCode Extension

Sekarang saya asumsikan kamu sudah berhasil menginstall vscode, step selanjutnya adalah install extension biar enak pas ngoding javascript.
klik `View > Extensions` atau menggunakan shortcut: `Ctrl + Shift + X`.

Cari extension biar ngodingnya enak:

1. [Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode) -> untuk memformat kodingan javascript kita.
2. [JavaScript (ES6) code snippets](https://marketplace.visualstudio.com/items?itemName=xabikos.JavaScriptSnippets) -> Code snippets for JavaScript in ES6 syntax
3. [ESLint](https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint) -> Integrates ESLint JavaScript into VS Code.

## Membuat folder belajar

Buatlah satu **folder baru** khusus untuk belajar javascript.
Disini saya akan memberi nama `belajar-javascript`.

Cara membuat folder lewat terminal windows

```bash
 mkdir belajar-javascript
```

Masuk ke folder lewat terminal windows

```bash
 cd belajar-javascript
```

Sekarang kita sudah punya semua kebutuhan untuk belajar, so mari kita mulai dengan belajar javascript dasar tahap 1.
