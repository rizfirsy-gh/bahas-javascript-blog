# Cara Belajar HTML dan CSS untuk Pemula

![Code](./code.jpg)

Sebagai orang yang mempelajari HTML dan CSS sendirian, saya dapat mengatakan bahwa tidak ada “cara terbaik” untuk mempelajari sesuatu secara umum. Cara saya melakukannya adalah menghabiskan sisa waktu saya setiap hari sepulang kuliah dan bekerja. Browsing di internet dan beli buku, dalam hal ini adalah HTML/CSS maka saya akan baca-baca di freecodecamp, W3Schools, dan MDN kemudian menuliskan semuanya yang sulit untuk saya pahami di kertas atau Microsoft Word(tapi sekarang lebih suka menggunakan Notion sih), kemudian saya coba membuat situs web sendiri. Tidak ada waktu untuk nongkrong atau main game. Bagi saya cara belajar seperti itu sangat berhasil, tetapi bagi kamu mungkin mengerikan wkwk.

{/* Gambarnya David Paschke di Unsplash */}

Pikirkan sebuah tujuan yang sangat spesifik. Mengapa ingin belajar HTML dan CSS? Apakah kamu ingin menjadi web developer? Apakah itu karena tugas sekolah dan kuliah? Apakah kamu ingin terlihat keren? Apakah kamu ingin merasa keren di tongkrongan?

Saya sendiri memiliki tujuan untuk bisa bikin website untuk portofolio desain grafis(saya memulai karir sebagai desain grafis). Tujuan kamu menentukan jalanmu, bro!

Jika kamu belum tahu apa yang harus dipelajari, dan bagaimana, freecodecamp dan w3schools adalah saran terbaik dari saya untuk belajar dasar koding melalui HTML dan CSS. Tetapi sebelum memulai sesuatu, tetapkan tujuan dan tantangan untuk dirimu sendiri.

Katakanlah kamu mencari situs web yang bagus di internet, dan kamu menemukan desain yang bagus. Saat mempelajari HTML dan CSS, cobalah mereplikasi desain itu dari nol, menggunakan apa yang kamu pelajari. Lihat apakah Kamu suka bekerja dengan HTML dan CSS.

Mungkin kamu ingin mendesain website, hanya menggunakan pulpen dan kertas karena kamu belum menguasai software desain(bagus kalo sudah bisa menggunakan Figma atau Adobe XD), jadi mulailah mendesain website tersebut di kertas bahkan sebelum memulai dengan HTML dan CSS.

{/* Gambarnya Hal Gatewood di Unsplash */}

Jika kamu belum tahu apa yang ingin kamu pelajari, dan bagaimana melakukannya, cara terbaik untuk memulainya adalah dengan mengerjakan proyek sambil belajar. Ambil beberapa screenshot situs web yang ingin kamu contek, atau download desainnya, dan mulailah membuatnya ke HTML/CSS versi kamu satu-persatu, pelan-pelan, per bagian.

Jika sudah terbiasa, terus biasakan diri untuk membuat beberapa website supaya kamu percaya diri dan kamu akan belajar cara membuat sesuatu dengan sangat cepat.

Setelah HTML dan CSS, kamu perlu belajar JavaScript supaya website kamu lebih dinamis. Gunakan metode yang sama ketika belajar HTML/CSS. Jangan hanya mempelajari JavaScript, mungkin sulit jika kamu tidak memiliki pengalaman programming tapi yakinlah ke diri sendiri kalo kamu bisa. The Power of Manifestation.

Gambarnya Randy Tarampi di Unsplash
Pada awalnya, proses belajarmu akan mudah. Tag html, bikin tabel, bikin form login, styling menggunakan css, bikin animasi lucu, tetapi kemudian akan mendapatkan hal-hal yang lebih sulit, dan jika tidak memiliki tujuan, kamu mungkin akan kecewa dan langsung menyerah. Maka kembali lagi, tentukan tujuan kamu belajar secara spesifik.

Setelah cukup menguasai HTML dan CSS, pikirkan aplikasi yang ingin kamu buat, dan mulailah mempelajari JavaScript sambil pelan-pelan tetap bikin website menggunakan HTML/CSS.

Kesimpulannya adalah beberapa kalimat yang saya tebalkan diatas. Coba perhatikan dan resapi kalimat-kalimat itu.

Good luck!
