Pseudo code adalah bahasa sederhana yang bisa manfaatkan untuk tidak memikirkan sintaks bahasa pemrograman dan hanya berfokus pada algoritma program.

<!--truncate-->

# Pseudo Code

Terdiri dari dua kata, pseudo yang artinya semu, code artinya kode.

### Fungsi pseudo code:

- Dapat digunakan sebagai alat dokumentasi
- Memudahkan kita menerjemahkan kalimat algoritma ke bahasa pemrograman
- Untuk mencari sebuah ide tanpa harus memikirkan implementasi dari sebuah ide dalam bahasa pemrograman
- Lebih mudah mengembangkan aplikasi yang dibuat

### Notasi pseudo code:

- INPUT: masukan
- OUTPUT: keluaran
- WHILE: digunakan pada program yang memiliki iterasi/pengulangan awal
- FOR: digunakan untuk perulangan/iterasi perhitungan
- REPEAT-UNTIL: digunakan untuk kondisi perulangan yang memiliki kondisi akhir
- IF-THEN-ELSE: digunakan untuk mengambil sebuah keputusan dari suatu kondisi

### Saran untuk menulis pseudo code:

- TUlis secara spesifik dan konsisten
- Gunakan indentasi
- Buat dengan sesederhana mungkin

### Struktur pseudo code:

- Judul algoritma
- Deklarasi: berisi keterangan seperti variable atau konstanta yang digunakan dalam algoritma
- Algoritma: proses atau langkah dari algoritma

### Contoh pseudo code:

Pseudo code untuk algoritma menjumlahkan dua angka:

```javascript
//Judul
Algoritma menjumlahkan dua angka

//deklarasi
variable angka1: integer //dibaca "buatlah variable dgn nama angka yang memegang value integer
variable angka2: integer
variable total: integer
//Algoritma
INPUT angka1
INPUT angka2
total = angka1 + angka2

```
